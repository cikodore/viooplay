# 🎬 ViooPlay

Modern Streaming Blogger Theme powered by TMDB.

> A fast, modern, Netflix-inspired Blogger theme with TMDB integration.

---

## Features

- 🎬 Dynamic Hero Banner
- 🔥 Trending Movies
- 🍿 Popular Movies
- ⭐ Top Rated Movies
- 📺 Popular TV Shows
- 🎞 Official Trailer
- 🔍 Live Search
- ❤️ Favorite Movies
- ⏯ Continue Watching
- 📱 Responsive Design
- 🌙 Dark Mode
- ⚡ Lazy Loading
- 🚀 Fast Performance
- 🎥 MP4 Player Support
- 💬 Subtitle Support (.vtt)
- 🔥 Infinite Scroll

---

## Tech Stack

- Blogger XML
- Bootstrap 5.3
- Vanilla JavaScript (ES6)
- SwiperJS
- TMDB API v3
- LocalStorage
- CSS Variables
